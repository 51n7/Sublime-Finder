# Sublime-Finder

### Shortcuts
| Command                           | Shortcut           |
| :-------------------------------- | :----------------- |
| Open Finder   | <kbd>⌃ + ⌘ + l</kbd> or <kbd>ctrl + shift + l</kbd> |
| Navigate Up   | <kbd>⌘+↑</kbd> or <kbd>ctrl+↑</kbd>   |
| Navigate Down | <kbd>⌘+↓</kbd> or <kbd>ctrl+↓</kbd>   |
| Open File | <kbd>⌘+↓</kbd> or <kbd>ctrl+↓</kbd>       |
